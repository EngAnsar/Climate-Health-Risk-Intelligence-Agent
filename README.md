## ğŸ§  Multi-Agent Architecture  
The system uses *coordinated AI agents*, each performing a specific task:

### *1. AQI Agent (API + Synthetic Hybrid)*
- Fetches AQI from AQICN API  
- Computes 30-day AQI trend  
- Identifies pollution spikes & sensitive-group risks  

### *2. Weather Stress Agent (API)*
- Fetches temperature, humidity, and heat index  
- Detects heatwave-like stress  
- Supports health-risk evaluation  

### *3. Green Cover Agent*
- Analyzes NDVI-like grid (synthetic satellite data)  
- Detects low-green areas  
- Clusters planning priority zones  

### *4. Waste Hotspot Agent*
- Processes 40+ synthetic citizen complaints  
- Categorizes issues  
- Identifies hotspots and high-frequency complaint zones  

### *5. Health Risk & Advisory LLM Agent*
- Uses Gemini/OpenAI LLM  
- Evaluates environmental stress impacts  
- Gives actionable recommendations for:  
  - Asthma patients  
  - Elderly  
  - Outdoor workers  
  - Children  

### *6. Action Planner Agent*
- Combines outputs of all agents  
- Computes â€œpriority scoreâ€ (impact Ã— feasibility)  
- Generates top 10 interventions  

### *7. Reporting Agent*
- Generates combined JSON output  
- Prepares AQI charts + NDVI heatmaps (PNG)  
- Optional: HTML report (Kaggle-safe)
